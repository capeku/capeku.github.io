About Rigel DX

I built this web app using adobe XD for the user interface and experience design , Vue.js for frontend and leveraged Google Firebase for backend and user authentication in a week. This app allows users register their pharmacies and also save and share medical test results as image files. This app was built to serve as an addon application for Rigel OS users.
